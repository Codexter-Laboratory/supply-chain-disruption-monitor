## Overview

This dashboard visualizes real-time system activity generated by the backend simulation engine, including ship movements, supply chain events, energy prices, and news updates.

Minimal React dashboard: **TanStack Query** for HTTP GraphQL, **graphql-ws** for subscriptions. Vite proxies `/graphql` (HTTP + WebSocket) to `http://localhost:3000`.

## Run

```bash
cd frontend
npm ci
npm run dev
```

Start the Nest API on port **3000** first. For live ticks and data volume, set `SIMULATION_ENABLED=true` on the backend.

## Architecture

- `src/services/api/` — GraphQL client, operation strings, typed fetch helpers (no React).
- `src/features/*/hooks/` — queries, subscriptions, cache updates.
- `src/features/*/components/` — presentational only (props in, JSX out).
- `src/app/` — shell, `QueryProvider`, `Dashboard` composition.

## Subscriptions

`useShipStatusSubscription` patches cached `ships` pages in place when `shipStatusChanged` fires (no full list refetch). `useSupplyChainEventSubscription` maintains a rolling feed for the sidebar.
